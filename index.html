Covid 19 European Map Visualisation Webapp

This is a webapp that has been designed to give European citizens the ability to visualise the current trends of Coronavirus throughout the continent. Both a map visualisation of the continent and a tabular representation of the data being represented is provided to the user.

Weekly Cases Weekly Deaths

Guide


Map Visualisation

  • The map has each country in Europe with colours that correspond to the metric that is being visualised, for example the new COVID-19 cases in that country per million in that country.
  • Upon hovering over a coloured country in the map with the cursor, a tooltip comes up which presents the name of the country and the value that is being represented by the graph.

Data Table

  • To the right of the map visualisation, there is a table that represents the data being displayed in the map in a tabular format.
  • By clicking on the column headings of the table, you can sort the table by the values of the column in both ascending and descending order.

Data Source


All of the data that is used in this web application comes from the Our World in Data Coronavirus source data, which is open source for all to use.
